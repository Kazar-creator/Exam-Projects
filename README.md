# Exam Preparation & Logic Exercises (PHP)

This repository contains a collection of small applications developed to master core programming concepts, object-oriented logic, and state management in PHP.

## Included Exercises

### 1. ATM Simulator (`/01-bancomat`)
* **Concepts**: Session handling, balance validation, PIN verification.
* **Logic**: Simulates withdrawals, deposits, and error handling for insufficient funds.

### 2. Digital Safe (`/02-cassaforte`)
* **Concepts**: State management, security logic.
* **Logic**: Simulates a safe that locks after 3 failed PIN attempts and requires a master key reset.

### 3. Vending Machine (`/03-macchinetta-caffe`)
* **Concepts**: Conditional logic, inventory management.
* **Logic**: Tracks available ingredients (coffee, milk, sugar) and calculates change based on coins inserted.
